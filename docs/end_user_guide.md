# Sundown VR Player – End User Guide

## Overview
The Sundown VR Player is a virtual reality application designed to deliver calming immersive 360° video experiences while collecting synchronized behavioral and physiological data for research purposes.

The system is intended for controlled research sessions and emphasizes minimal interaction, stable playback, and reliable data logging.

---

## Starting a Session
1. Power on the VR headset.
2. Launch the **Sundown VR Player** from the headset’s app library.
3. The application initializes the VR environment and prepares session logging.
---

## Controls & Interaction
- The UI is intentionally minimal
- Controls allow the user to:
  - Start video playback
  - Pause or stop a session
- User movement is allowed with locomotion or headmovent with a wrist UI if User is far away from Menu

---

## During the Experience
- Immersive 360° video playback fills the environment.
- Eye tracking, head pose, and interaction timing are continuously recorded.
- When enabled, physiological data from the smartwatch is streamed during the session.

---

## Ending a Session
- Sessions end automatically when playback finishes or manually via UI controls.
- Data logging stops at session end.
- Timestamped session files are saved locally for later review or transfer.

---

## Notes & Limitations
- This application is for **research and evaluation only**.
- It is not intended for clinical diagnosis or treatment.
- The experience is designed to be passive and non-interruptive.
